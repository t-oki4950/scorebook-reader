# 記入者別スタイルメモ

このファイルには、特定のスコアラーが使う書き癖や、画像から読み取りづらい記号の判断メモを蓄積します。

## 記録方針

- 断定できるルールは `symbol_dictionary.json`、`position_rules.json`、`compound_rules.json` に登録する。
- まだ曖昧な傾向や、画像の癖に依存する判断はこの Markdown に残す。
- 例: 「この記入者の二重✗は、2つ目の✗が薄く小さいことが多い」

## 未整理メモ

- 1つのマス目は1打席分として読む。マス目は実際の野球場に見立てられている。
- 中央の菱形はランナーがベースを回る進塁状況を表す。
- 正しい塁位置は、右下のコーナーが1塁、右上のコーナーが2塁、左上のコーナーが3塁。
- 中央の●または塗りつぶしは本塁生還・得点として読む。中央のlは残塁として読む。
- 打者の結果、特に出塁方法や打球方向は、右下の1塁側または中央の菱形内に書かれやすい。
- 例: 右下に⑧があり1塁までの線がある場合は、センター前ヒットの可能性が高い。
- 例: 右下に⑦があり2塁までの線または上側への到達がある場合は、レフト方向の二塁打の可能性がある。
